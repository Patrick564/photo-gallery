# Photo Gallery

A site for upload your photos.
[Deployed in Heroku](https://shot-gallery.herokuapp.com/)

## Installation

Install dependences

```bash
npm install
```

Build Tailwind CSS file

```bash
npm run build
```

## Usage

Run server with Node or Nodemon

```bash
# With Node
npm run start

# Or with Nodemon
nodemon server.js
```
